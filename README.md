# Install

1. Download

```bash
git clone https://github.com/landmaster135/dotfiles.git
cd dotfiles
```

2. Install

```bash
./install.sh
```

or with GUI(Hyprland/i3/sway setup)

```bash
./install.sh --gui
```

3. zsh plugin install

```bash
exec zsh
```

4. Enjoy!
